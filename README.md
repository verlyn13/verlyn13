# Jeffrey V. Johnson, Ph.D.

**Evaluations and agent-governance engineer**

I build systems that make AI-assisted work measurable, governable, and
auditable: evaluation evidence, explicit agent authority, and
evidence-preserving applied workflows.

I pair probabilistic AI tooling with deterministic, machine-checkable
verification.

## Current work

- **[Agentic-Coding Evaluation Lab](https://github.com/verlyn13/eval-lab-methodology):**
  a public methodology and independent verification surface for determining
  whether evidence justifies a model or coding-agent change. Current public
  proof is synthetic and makes no real-model performance claim.
- **Governance and agent-control architecture:** an anonymized private case
  study in delegated authority, evidence-gated controls, policy-as-code, and
  safe refusal.
- **Budget Triage:** a private financial evidence workbench that preserves
  provenance, exact monetary semantics, tenant boundaries, and human review.
  Private source. Sanitized architecture materials, synthetic evidence, and a
  guided technical walkthrough are available for relevant hiring
  conversations.

## Public implementation evidence

- **[Host Capability Substrate](https://github.com/jefahnierocks/host-capability-substrate):**
  a schema-first prototype for typed capabilities, authorization, provenance,
  and policy decisions for host-level agents. Runtime enforcement is not
  implemented.
- **[Agentic Architecture Audit](https://github.com/verlyn13/agentic-architecture-audit):**
